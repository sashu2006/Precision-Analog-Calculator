# Precision Analog Calculator

A purely continuous-time analog Arithmetic Logic Unit (ALU) built with LM358P op-amps and 1N4148 diodes, bypassing digital clocks and memory for instantaneous edge computation.

## Features

Our precision analog calculator implements the following core functionalities:

- **Addition** - Summing amplifier circuits for precise voltage addition
- **Subtraction** - Difference amplifier configurations for accurate subtraction
- **Multiplication** - Analog multiplier circuits for precise multiplication operations
- **Division** - Precision divider circuits for accurate division
- **Logarithm** - Log amplifier circuits for logarithmic transformations
- **Absolute Value** - Precision absolute value circuit with exceptional accuracy

## Performance Highlights

### Absolute Value Circuit
Our absolute value circuit demonstrated exceptional performance with:
- **Accuracy Error**: 0.95% within the working range
- **Input Range**: 1V to 12V
- **Performance**: Highly reliable and precise across the entire working range

## Technical Details

All circuits are implemented using:
- **Op-Amps**: LM358P operational amplifiers for precision analog processing
- **Diodes**: 1N4148 diodes for precision circuit operations
- **Architecture**: Fully continuous-time analog design with no digital clocks or memory storage
- **Response Time**: Instantaneous edge computation for real-time analog processing
